V   view 视图 （jsp html）

Spring MVC 应用于表现层的框架


SpringMVC 全名Springweb MVC 是一种基于java的实现MVC设计模型的请求驱动类型轻量级web框架 属于springframework的后续产品

SpringMVC本质可以认为是对servlet的封装 简化了servlet的开发
作用
1) 接受请求
2) 返回响应 跳转页面
